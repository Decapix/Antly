{% blocktrans %}La livraison{% endblocktrans %}

{% blocktrans %}La livraison de nos produits est assurée par des services de messagerie externes, incluant UPS, DPD, La Poste, Hermes, Royal Mail, Correos, et LnPost. Le choix du prestataire est déterminé par le vendeur responsable de votre commande, en fonction des tarifs des différents services de livraison. Il est important de noter que nous ne réalisons aucun bénéfice sur les frais de livraison. {% endblocktrans %}

{% blocktrans %}Pour les fourmis sensibles au froid, notamment durant les mois d'hiver, nous utilisons des chaufferettes pour garantir leur bien-être durant le transport. Le coût de ces chaufferettes est inclus soit dans le prix de l'article, soit dans les frais de livraison.{% endblocktrans %}

{% blocktrans %}Un courriel de confirmation est envoyé dès l'expédition de votre commande. Si vous n'avez pas reçu ce courriel dans la semaine suivant votre commande, veuillez nous contacter.{% endblocktrans %}

{% blocktrans %} Nous privilégions l'envoi des commandes en début de semaine pour éviter que les colis ne restent dans les entrepôts durant le weekend, réduisant ainsi le risque de mortalité pour les espèces les plus fragiles.{% endblocktrans %}

{% blocktrans %}Problème de livraison et retours{% endblocktrans %}
  • {% blocktrans %}En cas de décès partiel des fourmis durant le transport, aucun remboursement n'est accordé si des ouvrières survivent.{% endblocktrans %}
  • {% blocktrans %}Un remboursement partiel de la moitié de la valeur de la colonie est possible si la ou les reines sont vivantes tandis que toutes les ouvrières sont décédées.{% endblocktrans %}
  • {% blocktrans %}Un remboursement intégral, incluant les frais de port si la colonie était le seul produit commandé, est prévu uniquement en cas de décès total de la colonie.{% endblocktrans %}

{% blocktrans %}Pour chaque situation, une preuve sous forme d'image sera exigée. Ces politiques de remboursement s'appliquent uniquement durant la période estivale ; aucun remboursement pour les colonies décédées n'est effectué en hiver, conformément aux avertissements publiés sur notre site.{% endblocktrans %}

{% blocktrans %}Nous ne procédons à aucun remboursement pour les colonies décédées après les jours suivant la réception du colis.{% endblocktrans %}
